# setRoute( integer $Method, string $URI, function $function )

Set the routing for a given method/URI pair (or group of URI if regex is used). An example of usage can be: use Gishiki\Core;

Set the routing for a given method/URI pair (or group of URI if regex is used). An example of usage can be:

use Gishiki\Core;

Core\Routing::setRoute(Core\Routing::GET, "/user/{id}", function ($params) {
     foreach ($params as $param_name => $param_value) {
         echo $param_name." => ".$param_value;
     }
});

//if the framework is installed at https://site.com/ when an user requests
//https://site.com/user/5476 the user will see: id => 5476

Be aware that the passed parameter is an object instance from \Gishiki\Algorithms\CyclableCollection if regex routing is used, otherwise it is NULL

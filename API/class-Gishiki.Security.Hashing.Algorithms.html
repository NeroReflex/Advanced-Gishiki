public static string
# hash( string $message, string $algorithm = Gishiki\Security\Hashing\Algorithms::MD5, boolean $binaryUnsafe = false )

Generate the message digest for the given message.

Generate the message digest for the given message.

An example usage is:

$message = "this is the message to be hashed";

$test_gishiki_md5 = Algorithms::hash($message, Algorithms::MD5);
$test_php_md5 = md5($message);

if ($test_gishiki_md5 == $test_php_md5) {
    echo "Gishiki's MD5 produces the same exact hash of the PHP's MD5";
}

public static string
# pbkdf2( string $password, string $salt, string $keyLength, string $count, string $algorithm = Gishiki\Security\Hashing\Algorithms::SHA1, string $binaryUnsafe = false )

PBKDF2 key derivation function as defined by RSA's PKCS #5: https://www.ietf.org/rfc/rfc2898.txt.

PBKDF2 key derivation function as defined by RSA's PKCS #5: https://www.ietf.org/rfc/rfc2898.txt.

Test vectors can be found here: https://www.ietf.org/rfc/rfc6070.txt

This implementation of PBKDF2 was originally created by https://defuse.ca With improvements by http://www.variations-of-shadow.com

Parameters

$password
the password
$salt
a salt that is unique to the password
$keyLength
the length of the derived key in bytes
$count
iteration count. Higher is better, but slower. Recommended: At least 1000
$algorithm
the hash algorithm to use. Recommended: SHA256
$binaryUnsafe
if true, the key is returned in raw binary format. Hex encoded otherwise

setDatabaseAutoUpdate()

setDatabaseAutoUpdate(boolean  $value)

Set a new value to the auto-update database operation.

When the auto-update is on (has been manually set to true) the database is updated before running the application.

Running an application is done by calling the run() function, so this function must be called before the run() call.

Parameters

boolean $value

the new value

Remove objects

In a product, may it be a webpage, a website or a webapp; it is important to be able to delete one or more objects already stored. To achieve the result of deleting elements, the ORM gives you a function called Remove.

The Remove function works exactly as the Restore function. The Remove function returns the number of removed objects: 
class Book_Controller extends Gishiki_Controller {
    function RemoveBooksICannotSell()
    {
        //prepare the selection of objects to be removed
        $conditions = new Clauses();
        $conditions->AddClause("quantity", Criteria::EQUAL, 0);

        //restore every book fulfilling each given clause
        $books = $this->Remove("Book", $conditions);

        //work with this book from the controller
        return $books;
    }
}
Extremely caution: if no clauses are added to the list of clauses the entire table OR its entire content will be removed (it depends on the selected RDBMS).

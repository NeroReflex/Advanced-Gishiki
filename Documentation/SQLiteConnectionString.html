SQLite v2 and v3 Connection Strings

Database connections are explained in the next chapter.
In the next chapter you will read about a function, called CreateConnection, that connects a model to a database using a given connection string.
This paragraph focuses on how to write a SQLite connection string.

A SQLite connection string is written as: 
$connString = "sqlite://Database";
The connection string contains the database type, in this example 'sqlite', and the database name.

Allowed database types are:
  • sqlite and sqlite2 for a SQLite v2 connection
  • sqlite3 for a SQLite v3 connection
To use a SQLite v2 database you have to install pdo and sqlite driver for pdo.
To use a SQLite v3 database you have to enable the sqlite3 php extension.
SQLite v3 databases support password. To use a password with your database you have to modify your connection string to: 
$connString = "sqlite://Database:my secret p@ssword";

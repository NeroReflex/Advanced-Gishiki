SRDBMS Server-based Connection Strings

Database connections are explained in the next chapter.
In the next chapter you will read about a function, called CreateConnection, that connects a model to a database using a given connection string.
This paragraph focuses on how to write a connection string to a server-based RDBMS.

A server-based RDBMS connection string is written as: 
$connString = "mysql://dbname@host:post/username:password";
The connection string contains the database type, in this example 'mysql', the database name, the database host, the database port (default if not given), the name of the user used to access the database and the password for that user.

Allowed database types are:
  • mysql for a MySQL/MariaDB connection
  • firebird for a Firebird connection
  • pgsql for a PostgreSQL connection
  • firebird for a Firebird connection
To use any of those database types you have to install the pdo extension and the proper driver for pdo.
